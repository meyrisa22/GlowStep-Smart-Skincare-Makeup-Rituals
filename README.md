# GlowStep-Smart-Skincare-Makeup-Rituals
A luxury-designed web application to organize skincare and makeup routines with a built-in 14-day smart scheduler that protects skin barrier from over-exfoliation.

## ✨ Key Highlights

### 🌿 Smart Skincare Planner
Bukan sekadar daftar, tapi asisten kesehatan kulit:
* **Dynamic 14-Day Calendar**: Membuat jadwal otomatis selama 2 minggu.
* **Barrier Protection Logic**: Mendeteksi produk eksfoliasi (Peeling, Exfo Toner) dan masker agar tidak digunakan setiap hari guna mencegah iritasi.
* **AM/PM Smart Filter**: Memisahkan secara otomatis mana produk untuk perlindungan siang (Sunscreen) dan pembersihan/nutrisi malam (Night Cream, Cleansing Oil).

### 💄 Professional Makeup Layering
Urutan presisi untuk hasil *flawless*:
* **Logical Prep & Prime**: Memastikan produk dasar (Primer/BB Cream) berada di urutan pertama sebelum produk dekoratif.
* **Step-by-Step Visualization**: Memberikan panduan urutan dari *complexion*, *eye makeup*, hingga *finishing spray*.
* **Inventory Based**: Urutan hanya dibuat berdasarkan produk yang kamu miliki saat ini.
